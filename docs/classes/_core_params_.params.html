Class Params

The Params class is responsible for all transforms applied to an Aspect scene description. It transforms a flat object into a Scaffold (a table of rows, one per key). Methods transform data in this table, convert it to flat object and then to a new Scaffold.

Its constructor receives a scene object and internally creates a new Scaffold from it. This class is hygenic. That means that every method applies a transformation to the classes current data and supplies this to a new Params instance. This design ensures that Params instances are functionally stateless.
