Class Scaffold

Scaffold is a class which wraps and extends data-forge DataFrames. It's API is conformed to be more like Pandas DataFrames and should be thought of as a TypeScript equivalent.

  • to_lut(column: string): object

  • Converts Scaffold into lut object (lookup table). Unique value of given column are cast as keys Rows as Arrays of objects are cast as values per key

    Parameters

    • column: string

      Column to group by

    Returns object

    Lut object

    Example 
    >>> data = new Scaffold().from_array([
    {a: 1, b: 10, c: 100},
    {a: 2, b: 20, c: 200},
    {a: 2, b: 30, c: 300},
]).to_lut("a");
{
    1: [
           {a: 1, b: 10, c: 100}
       ],
    2: [
           {a: 2, b: 20, c: 200},
           {a: 2, b: 30, c: 300},
       ]
}
